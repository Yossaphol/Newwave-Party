NEW WAVE

พรรค NEW WAVE ขอบคุณเป็นอย่างสูงที่มีความสนใจในพรรค NEW WAVE พรรคของเราเตรียมตัวมากว่า1ปีแล้วตั้งแต่ยังเป็นกลุ่มเล็ก ๆ จนกลายเป็นพรรคที่สมบูรณ์ พวกเราคือตัวแทนของนักเรียนที่จะมาเปลี่ยนแปลงโรงเรียน คิดจริง ทำจริง ไปให้สุด พรรค NEW WAVE เพียบพร้อมในทุกด้านไม่ว่าจะเป็นกีฬา เกม ดนตรี วิชาการ และสวัสดิการ

ทำไมถึงชื่อพรรค NEW WAVE ?

     NEW WAVE คือพรรคที่เกิดขึ้นจากกลุ่ม 3 กลุ่ม คือ 1. กลุ่มนักเรียนชาย 2. กลุ่มนักเรียนหญิง 3. กลุ่ม LGBTQIA+ แสดงให้เห็นถึงการอยู่ร่วมกันอย่างมีความสุขในรั้วส้มฟ้า สัญลักษณ์ที่ชูให้เห็นถึงการอยู่ร่วมกันคือคลื่น 3 ลูกที่จะชะแผ่นดินให้งดงาม ลุกขึ้นมาเปลี่ยนแปลงโรงเรียน ไม่เลือกเดินตามทางของผู้มีอำนาจ ไม่เลือกที่จะเงียบแล้วรอให้เหตุการณ์ผ่านไป คลื่น 3 ลูกนี้ จะรวมเป็นหนึ่งเดียวแล้วซัดเข้าชายฝั่ง แสดงให้เห็นถึงความสามัคคีของพรรค NEW WAVE ถึงเวลาแล้วที่เราทุกคนจะต้องลุกขึ้นมาเปลี่ยนแปลงโรงเรียนชลราษฎรอำรุงให้ก้าวไปสู่ยุคใหม่ที่ดียิ่งกว่าเดิม

“ทุกความสำเร็จเริ่มต้นที่ตัวเรา แล้วมาก้าวไปด้วยกันนะครับ”

ทำไมต้องเลือกพรรค

NEW WAVE to NEW SCHOOL

     พรรค NEW WAVE มีผู้มากประสบการณ์หลากหลายด้าน ไม่ว่าจะเป็นด้านวิชาการ หัวหน้าพรรค NEW WAVE ณภัทร พูลเหลือ (ออมสิน) และรองหัวหน้าพรรคคนที่ 1 วัชรพล ยูถะสุนทร (ปูไข่) เป็นผู้ที่ผ่านสนามการแข่งขันทางวิชาการมามากมาย มีความเข้าใจในระบบและปัญหาของการแข่งขันเป็นอย่างดี

      นอกจากนี้ในด้านกิจกรรม ชยธร อุดมทองรัตน์ (ธร) รองหัวหน้าพรรคคนที่ 2 เป็นผู้ที่ทำงานจิตอาสามามากมายและยังเคยช่วยงานสภานักเรียนปี 66 เป็นบางครั้งบางคราวแม้จะไม่ได้เป็นสมาชิกสภานักเรียน ธณภรรษ ชะอุ่ม (โมเน่) หัวหน้าฝ่ายประชาสัมพันธ์ เป็นผู้ที่มีความสามารถในวาทศิลป์ และมีความสามารถในด้านเทคโนโลยี การตัดต่อ ในด้านการเงิน เหรัญญิกพรรค ลฎาภา สุวรรณ์ปุสิก (น้ำอิง) ผู้มีความสนใจในด้านเศรษฐศาสตร์ตลอดจนความสามารถในการพูดภาษาอังกฤษอย่างคล่องแคล่ว ไม่เพียงเท่านี้พรรค NEW WAVE ยังส่งเสริมกิจกรรมทั้งการกีฬาและเกม เรามี นพกร หินจีน (โอ๋) เป็นหัวหน้าฝ่ายกิจกรรม โอ๋คือหนึ่งในนักเรียนที่มีความคุ้นเคยกับสนามกีฬาและโดมอเนกประสงค์ทำให้ทราบถึงปัญหาของสถานที่และการกีฬาเป็นอย่างดี และปณิธิ ศิริตันหยง (อิม) นักแข่งเกม RoV เป็นโพรเพลเยอร์ในด้านเกมโดยเฉพาะ

      ในด้านของประสบการณ์ แม้พรรค NEW WAVE จะมีสมาชิกที่เคยเป็นสมาชิกสภานักเรียนมาก่อนไม่มากนัก แต่ก็ไม่อาจพูดได้ว่าเราไร้ซึ่งประสบการณ์ พรรคของเราก่อตั้งขึ้นในวันที่ 3 พฤษภาคม พ.ศ. 2566 โดย วัชรพล ยูถะสุนทร (ปูไข่) อดีตหัวหน้าพรรค NEW WAVE และอดีตหัวหน้าฝ่ายวิชาการของสภานักเรียนประจำปีการศึกษา 2566 เราบางคนอยู่ที่ชลชายมา 4-5 ปี เราเห็นถึงปัญหาแต่ก็ทำอะไรไม่ได้ เสียงของพวกเราส่งไปไม่ถึง จึงเป็นเหตุให้พรรค NEW WAVE ถือกำเนิดขึ้น เพื่อทำให้เสียงของคนตัวเล็กเป็นเสียงที่ยิ่งใหญ่

เสรีภาพ เสมอภาค ภราดรภาพ

(1) พรรค NEW WAVE เชื่อมั่นในการเมืองยุคใหม่ สังคมใหม่ และโรงเรียนแบบใหม่

(2) พรรค NEW WAVE เชื่อมั่นว่าบุคคลทุกคนมีความเสมอภาคและเท่าเทียมกัน เคารพศักดิ์ศรีความเป็นมนุษย์ เคารพความแตกต่างทางความคิด ความหลากหลายทางเพศ และเคารพถึงความหลากลายในด้านต่าง ๆ

(3) พรรค NEW WAVE เชื่อมั่นว่าสังคมที่ดีต้องมี เสรีภาพ เสมอภาค และภราดรภาพ

ปัญหาซ้ำซ้อนในโรงเรียน ?

     ในยุคที่โรงเรียนต้องพบกับปัญหาซ้ำซ้อน หรือการมองเห็นสิ่งที่เกิดขึ้นแต่กลับไม่สามารถต่อต้านหรือเปลี่ยนแปลงได้ สอดคล้องกับการแก้ปัญหานี้โดยนโยบาย “ความเงียบที่ดังที่สุด”

     พรรค NEW WAVE ไม่สามารถทนมองในสิ่งที่ไม่ถูกต้องได้ จึงสร้างนโยบายไว้ทั้งสิ้น 69 นโยบาย เพื่อเปลี่ยนแปลงโรงเรียนให้ถึงที่สุด 69 นโยบายนี้ ครอบคลุมทุกปัญหาและได้ประเมินมาแล้วว่าจะสามารถทำได้ไหม แก้ได้ไหม ที่เป็นคำถามในใจทุกครั้งเมื่อถึงวันหาเสียง วันเลือกตั้งของคนหลาย ๆ คน ไม่ต้องห่วงครับ พรรค NEW WAVE ทราบถึงข้อนี้ดี ด้วย 69 นโยบายนี้ พรรคเราจะร่วมแก้ไขปัญหา ผลักดันนโยบายให้สำเร็จเพื่อเป็นจุดเริ่มต้นครั้งใหม่ที่ดียิ่งขึ้นเพื่อนักเรียนรุ่นหลังต่อไป

การสืบค้นข้อมูลในโรงเรียน

     พรรค NEW WAVE ได้ทำการค้นคว้าข้อมูลของนักเรียนแต่ละคน เพื่อทำการคิดค้นวิธีการแก้ไขปัญหา โดยมีผู้ลงแสดงความคิดเห็นมากกว่า 100 คนใน 1 สัปดาห์ เพื่อเรียกร้องให้แก้ไขปัญหาต่าง ๆ ในโรงเรียน

เลือกเราแล้วจะติดใจ

     ไม่เพียงแต่แก้ไขยังมีการเสนอเพิ่มเติมเกี่ยวกับการเพิ่มกิจกรรมใหม่เข้ามาในโรงเรียนอีกด้วย ทางพรรคจึงร่วมกันวางแผนนโยบายต่าง ๆ ตามการเรียกร้องของพี่ ๆ น้อง ๆ ชาวชลชาย เช่น CRU OPEN HOUSE , CRU INSPIRE , CRU CONTEST ,CRU SPOTLIGHT ฯลฯ เพื่อลดปัญหาความเครียด เพิ่มความสามัคคี ให้นักเรียนทุกคนได้มีส่วนร่วมในกิจกรรม ได้แสดงความสามารถต่าง ๆ เปิดโอกาสให้ผู้ที่มีความสามารถพิเศษได้นำความสามารถนั้นออกมาแสดง รวมถึงการปรับปรุงกิจกรรมเดิม อย่างเช่น CRU MUSIC WEEK ให้เป็นไปตามเสียงส่วนมาก ส่งเสริมการเปลี่ยนป้ายแผ่นหลังของโรงเรียนเป็นจอ LED ผลักดันการเบิกงบประมาณเครื่องดนตรีโรงเรียน และทางพรรค NEW WAVE ยังเน้นย้ำถึงปัญหาภายในโรงเรียนจึงได้ร่วมกันคิดไอเดียในการแก้ปัญหาต่าง ๆ ไม่ว่าจะเป็นการลดขยะ ภายในโรงเรียน เพิ่มพื้นที่สีเขียว ตามนโยบายสนับสนุน CRU ZERO WASTE ของพรรค NEW WAVE ที่แก้ไขเกี่ยวกับพลาสติกในโรงเรียน การเพิ่มต้นไม้ในโรงเรียน ที่มีกิจกรรมให้นักเรียนของเราได้ร่วมเล่น เช่น การนำขวดน้ำมาใช้ในโรงเรียนอย่างสม่ำเสมอจะได้รางวัลสุดพิเศษจากท่านผู้อำนวยการของโรงเรียน รวมถึงนโยบายนี้ยังได้รับความร่วมมือจากคุณครูและบุคลากรในโรงเรียน อีกทั้งพรรคของเรายังสนับสนุนนโยบายห้องเรียนสีขาวเพื่อให้โรงเรียนของเราเกิดความสงบรวมไปถึงพรรคของเรายังมีหลักการที่ว่า

“ห้องน้ำสะอาด

สภาพแวดล้อมดี

พื้นที่ปลอดภัย”

     เรื่องห้องน้ำ เป็นสิ่งที่สำคัญมาก ทางพรรคเราคิดนโยบายเกี่ยวกับห้องน้ำในโรงเรียนไว้มากมายที่ทำได้จริง เป็นขั้นตอน เช่น การติดตั้งอุปกรณ์ดับกลิ่นภายในห้องน้ำ การปรับปรุงห้องน้ำ เปลี่ยนรางปัสสาวะเป็นโถปัสสาวะ การเปลี่ยนหัวก๊อกน้ำ หรือการซ่อมบำรุงบานประตูที่ชำรุดเสียหาย

      แต่ยังคงมีสายฉีดชำระอยู่มีถังขยะไว้แต่ละห้องน้ำ สร้างราวไว้แขวนของ ผลักดันนโยบายการเปลี่ยนแปลงห้องน้ำโรงเรียนชลราษฎรอำรุงให้เป็นรูปแบบใหม่ การปรับปรุงห้องน้ำโดม เช่น ติดหลอดไฟ ซ่อมบำรุง เป็นต้น

     การเปลี่ยนแปลงห้องน้ำหญิงบริเวณโดมเพื่อให้มีความปลอดภัยมากขึ้น สิ่งเหล่านี้จะทำให้คุณภาพห้องน้ำโรงเรียนเป็นไปตามแนวทางที่ดีขึ้น สะท้อนการแก้ปัญหาของโรงเรียนให้ตรงจุด

อุบัติเหตุในโรงเรียน ?

     ปัญหารถในโรงเรียนเพื่อให้โรงเรียนเป็นพื้นที่ปลอดภัยสำหรับนักเรียนทุกคน

     นโยบายการติดสติกเกอร์นักเรียนชลชายสำหรับผู้ที่ลงทะเบียนกับโรงเรียน โดยนำมาติดไว้ที่รถหรือหมวกนิรภัย เพื่อให้ทราบว่าเป็นนักเรียนเพื่อป้องกันการเกิดอุบัติเหตุ หรือการเพิ่มพื้นที่จักรยานในโรงเรียนสนับสนุนการลดคาร์บอนไดออกไซด์ ตามแนวคิด ZERO WASTE ลดโลกร้อน ผู้ที่เป็นช่างซ่อมบำรุงเมื่อนำรถเข้ามาในโรงเรียน จะถูกแจ้งให้กับทราบผ่านเสียงตามสาย เพื่อความปลอดภัยที่ดียิ่งขึ้น ผลักดันให้สามารถขับขี่พาหนะในสถานศึกษาได้ไม่เกิน 40 กิโลเมตรต่อชั่วโมง เพื่อป้องกันอุบัติเหตุทางถนนและยังมีอื่น ๆ อีกมากมายติดตามได้ที่ หน้านโยบาย

      ไม่เพียงแต่การแก้ไข แต่ยังคงเพิ่มสวัสดิการสำหรับนักเรียนชลราษฎรอำรุงโดยเฉพาะบุคลากรในโรงเรียนอีกด้วยเช่น การร้องเรียนปัญหาต่าง ๆ ผ่านทางเพจ NEW WAVE official เพื่อดำเนินการแก้ปัญหาแบบเร่งด่วน ตามคำเรียกร้องของนักเรียนของทุกคน หรือการเป็นที่ปรึกษาปัญหาต่าง ๆ ให้กับนักเรียนทุกคนโดยจะดำเนินการร่วมกับงานแนะแนวเพื่อแก้ปัญหาของเพื่อน ๆ และเป็นที่พึ่งพาสำหรับทุกคน พรรค NEW WAVE ยังมีสวัสดิการให้กับทุกคนมากกว่า 20 นโยบาย

เมื่อได้รับตำแหน่งคณะกรรมการนักเรียนปีการศึกษา 2567 สุดท้ายนี้พรรค NEW WAVE ยังมีนโยบายอีกมากมาย นโยบายที่พร้อมจะนำมาเสนอให้กับทางโรงเรียนสู่สังคมที่ยั่งยืน

รูปการดำเนินงานในแต่ละนโยบาย

     แผนงาน การจัดการนโยบายแต่ละนโยบาย แบบสรุปความ

1. ได้รับการอนุมัติในที่ประชุมพรรค NEW WAVE ผ่านคณะกรรมการบริหารพรรค

2. ปรึกษาครูที่ปรึกษาพรรค NEW WAVE ถึงการปฏิบัติและแผนงาน

3. นำมาเป็นนโยบายพรรค NEW WAVE ตามประเภทต่าง ๆ

4. เมื่อได้รับตำแหน่งสภานักเรียนจะทำการเสนอนโยบายและแผนงาน ให้กับทางโรงเรียนโดยทันที

5. เมื่อสำเร็จจะนำมาใช้โดยทันที โดยอยู่ภายใต้การดูแลของพรรค NEW WAVE และถูกควบคุมภายใต้การดูแลของโรงเรียนและครูทุกคนในโรงเรียน

พรรค NEW WAVE คือกลุ่มของตัวแทนทั้ง 3 กลุ่ม ในโรงเรียนชลราษฎรอำรุงที่มารวมเป็นปึกแผ่นเดียวกัน
